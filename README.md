# PHP 웹솔루션

저가형 호스팅 환경에 최적화된 경량 PHP CMS 솔루션입니다.

## 주요 기능

- ✅ **사용자 관리**: 회원가입, 로그인, 프로필 관리
- ✅ **게시판 시스템**: 다중 게시판, 게시글, 댓글 관리
- ✅ **컨텐츠 관리**: 정적 페이지 생성 및 관리
- ✅ **관리자 페이지**: 통합 관리 대시보드 (/admin)
- ✅ **플러그인 시스템**: 확장 가능한 훅 기반 아키텍처
- ✅ **CKEditor 지원**: 4.22.1 오픈소스 버전

## 시스템 요구사항

- PHP 7.4 이상
- MySQL 5.7 이상
- Apache with mod_rewrite
